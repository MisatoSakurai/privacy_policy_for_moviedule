# Moviedule プライバシーポリシー

このリポジトリは [Moviedule](https://github.com/your-username/moviedule_python) アプリのプライバシーポリシーページです。

## GitHub Pages で公開する手順

1. **リポジトリを GitHub に作成する**
   - 新規リポジトリを作成（例: `privacy_for_moviedule` または `privacy-moviedule`）
   - このフォルダの内容（`index.html` と `README.md`）をプッシュ

2. **GitHub Pages を有効にする**
   - リポジトリの **Settings** → **Pages**
   - **Source**: "Deploy from a branch" を選択
   - **Branch**: `main`（または `master`）、**Folder**: `/ (root)` を選択
   - Save をクリック

3. **公開 URL**
   - 数分後、次の URL で表示されます:  
     `https://<あなたのユーザー名>.github.io/<リポジトリ名>/`  
   - 例: `https://sakuraimisato.github.io/privacy_for_moviedule/`
